# グループチャット機能のテスト手順

## 手動テスト項目

### 1. 基本機能テスト

#### 1.1 グループチャットモードの切り替え
- [ ] ヘッダーのグループアイコンをクリックしてグループモードに切り替わることを確認
- [ ] グループモードでGroupChatInterfaceが表示されることを確認
- [ ] 通常モードに戻れることを確認

#### 1.2 グループセッション作成
- [ ] 「グループチャットを開始」ボタンが表示されることを確認
- [ ] 2人以上のキャラクターが選択可能であることを確認
- [ ] グループ名を設定できることを確認
- [ ] チャットモード（順次、同時、ランダム、スマート）を選択できることを確認
- [ ] グループセッションが正常に作成されることを確認

#### 1.3 メッセージ送受信
- [ ] グループセッションでメッセージが送信できることを確認
- [ ] キャラクター名がメッセージバブルに表示されることを確認
- [ ] 複数キャラクターの応答が正常に表示されることを確認

### 2. UI表示テスト

#### 2.1 グループチャットヘッダー
- [ ] アクティブなグループセッション情報が表示されることを確認
- [ ] 参加キャラクター数が正しく表示されることを確認
- [ ] チャットモードが表示されることを確認

#### 2.2 MessageBubble
- [ ] グループチャットでキャラクター名が表示されることを確認
- [ ] 通常チャットで名前が表示されないことを確認
- [ ] アバター画像が正しく表示されることを確認

### 3. エラーハンドリング

#### 3.1 無効なセッション
- [ ] キャラクターが1人以下の場合、エラーメッセージが表示されることを確認
- [ ] セッション作成時のエラーが適切に処理されることを確認

#### 3.2 状態管理
- [ ] グループモードと通常モードの切り替えが正常に動作することを確認
- [ ] セッション状態が正しく保持されることを確認
